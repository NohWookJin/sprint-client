# SPRINT

자기개발을 좋아하는 사람을 위한 루틴 관리 서비스 SPRINT입니다. 🏃

👉 [서비스 링크](https://www.sprints.co.kr)에서 반복하는 루틴을 만들고 제공되는 분석을 통해 성장해보세요. (운영 종료)

## How to Start
1. 의존성 설치
```
node install
```

2. 실행(로컬)
```
npm run dev  
```
